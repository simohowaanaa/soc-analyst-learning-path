# Introduction to SOC

Leçon d'orientation du cours SOC Fundamentals — pas de contenu technique, elle explique
comment aborder le cours.

## Ce que couvre le cours (annoncé ici)

- La structure d'un SOC (Security Operations Center)
- Le fonctionnement (opération) d'un SOC
- Les outils/produits SOC
- Comment un analyste SOC doit utiliser ses outils
- Les erreurs courantes commises par les analystes SOC

## Conseils selon le profil

- **Futur SOC Analyst** : se concentrer sur les outils SIEM, Log Management, EDR, et prendre
  des notes. Une fois ce cours terminé, possibilité de passer à des cours plus techniques —
  mais il faut commencer par celui-ci.
- **SOC Analyst déjà en poste** : sections "Quick Tips" + exemples d'erreurs réelles à étudier
  pour améliorer les bonnes pratiques.

## Recommandation

Suivre le cours dans l'ordre (tout le contenu est lié), quitte à l'étaler sur plusieurs jours/semaines.
Ressource bonus : vidéo "How to Become a SOC Analyst?" (LetsDefend).
