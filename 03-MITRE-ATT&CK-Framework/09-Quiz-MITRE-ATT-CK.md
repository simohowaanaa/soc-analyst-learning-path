# Quiz de validation — MITRE ATT&CK Framework

Quiz final de la section. Résultat obtenu sur la plateforme : **14 points**.

**Q1. What is the concept that expresses the motivation for the action of the cyber
attacker within the MITRE ATT&CK Framework?**
Réponse : **Tactic**
→ Exprime la motivation/le "pourquoi" de l'action de l'attaquant
([leçon 4](04-Tactics.md)).

**Q2. What is the concept that explains how the cyber attacker performs his action within
the MITRE ATT&CK Framework?**
Réponse : **Technique**
→ Explique "comment" l'attaquant réalise son action
([leçon 5](05-Techniques-and-Sub-Techniques.md)).

**Q3. What is the concept that expresses the application examples of the method used by the
cyber attacker within the MITRE ATT&CK Framework?**
Réponse : **Procedure**
→ Exemples concrets d'application d'une technique dans la vraie vie
([leçon 5](05-Techniques-and-Sub-Techniques.md)).

**Q4. Which of the following is an ID of a technique in the MITRE ATT&CK Framework?**
Réponse : **T1426**
→ Format d'ID technique = `T` + 4 chiffres. `TA00xx` = tactique, `M10xx` = mitigation,
`S0xxx` = software.

**Q5. Which of the following enterprise techniques belongs to a different tactic?**
Réponse (confirmée ✓) : **Exploit Public-Facing Application**
→ Les 3 autres (Phishing for Information, Active Scanning, Gather Victim Identity
Information) sont sous la tactique **Reconnaissance** ; celle-ci est sous **Initial Access**.

**Q6. In which matrix is the "Impair Process Control" tactic located?**
Réponse (confirmée ✓) : **ICS Matrix**
→ Tactique spécifique au monde industriel ([leçon 4](04-Tactics.md)).

**Q7. What is the tool used for credential dumping specifically only for Linux systems by
the "TeamTNT" APT group?**
Réponse (confirmée ✓) : **MimiPenguin**
→ Équivalent Linux de Mimikatz, utilisé par le groupe TeamTNT.
